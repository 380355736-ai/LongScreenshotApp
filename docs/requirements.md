# 需求规格说明书 — iOS 截长图 App

## 项目概述

**项目名称**：截长图 (LongScreenshot)
**目标平台**：iOS 15.0+
**目标设备**：iPhone
**目标用户**：需要截取长图的普通用户

## 核心需求

### 1. 屏幕录制
- 使用 iOS 原生 ReplayKit 录制屏幕
- 用户可切换到任意 App 滚动浏览
- 录制状态有明确视觉反馈

### 2. 自动拼接
- 录制完成后自动提取视频帧
- 使用 Vision 框架检测相邻帧重叠区域
- 自动拼接为一张长图
- 裁剪左右黑边，输出干净结果

### 3. 长图预览
- 拼接完成展示结果预览
- 支持缩放和滚动查看
- 显示图片尺寸信息

### 4. 保存与分享
- 一键保存到系统相册
- 支持系统分享菜单（微信/QQ/AirDrop 等）

### 5. UI 设计
- 简洁直观，iOS 原生风格
- 蓝色主色调
- 录制按钮显眼易操作
- 处理过程有进度反馈

---

## 非功能性需求

| 类别 | 要求 |
|------|------|
| **性能** | 10 秒视频拼接时间 < 5 秒 |
| **内存** | 拼接过程中内存 < 200MB |
| **最低版本** | iOS 15.0 |
| **权限** | 屏幕录制、相册写入 |

---

## 不包含的功能
- 视频录制保存
- 网页截长图（非录制模式）
- 图片编辑/标注
