# 分阶段开发计划

## 开发原则
1. **每阶段只做一件事**，完成验证后再进入下一阶段
2. **每个阶段产出可运行/可验证的代码**
3. **每阶段结束记录开发日志**
4. **先在 Windows 编写代码，再通过 GitHub Actions 编译验证**
