# UI 设计规范 — iOS 截长图 App

## 设计原则

- **iOS 原生感**：遵循 Apple Human Interface Guidelines
- **简洁直观**：主屏幕只有一个核心操作按钮
- **反馈明确**：每个步骤都有视觉反馈

## 色彩

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `iOS Blue` (#007AFF) | 按钮、进度、强调 |
| 录色 | `iOS Red` (#FF3B30) | 停止按钮、录制指示 |
| 背景 | `systemGroupedBackground` | iOS 系统分组背景 |
| 文字 | `primary` / `secondary` | 系统自适应文字色 |

## 页面流程

```
┌──────────┐     ┌──────────┐     ┌──────────┐     ┌──────────┐
│  首页     │ →  │  录制中   │ →  │  处理中   │ →  │  结果页   │
│          │     │          │     │          │     │          │
│  📱图标   │     │  🔴 红点   │     │  ⏳ 进度   │     │  长图预览  │
│  开始录制  │     │  停止按钮  │     │  步骤文字  │     │  保存分享  │
│          │     │          │     │          │     │          │
└──────────┘     └──────────┘     └──────────┘     └──────────┘
```

## 字体

- 标题: System Font, Large Title
- 正文: System Font, Body/Subheadline
- 按钮: System Font, Title3, Semibold
- 提示: System Font, Caption

## 交互

| 操作 | 反馈 |
|------|------|
| 点击开始录制 | 按钮变红、脉冲动画 |
| 录制中 | 按钮显示"停止录制"、红点闪烁 |
| 停止录制 | 跳转处理页、进度条动画 |
| 拼接完成 | 跳转结果页、可缩放预览 |
| 保存成功 | 系统 Alert "已保存" |
